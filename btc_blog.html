Project scope and explanation

This project was really just a fun way to practice and explore some more machine learning (ML) and reinforcement learning (RL) in python. I also wanted to push my coding further by incorporating an API into the project and aiming to build a usable app (an MVP at least if not a full app haha). It is a work in progress, I will continue to update this page as I make progress!


I broke down the project into bite sized chunks, namely:

  1. Preliminary data exploration
  2. Selection of ARIMA parameters for predictions
  3. Generation of predictions for training and testing in RL environment
  4. Build RL training environment
  5. Train BTC trading bot in RL environment (I foresee lots of frustration at this step haha)
  6. Build beginning of application that uses an API to grab the latest BTC prices and makes a prediction for the future.
  7. Integrate the trained Trading bot into the application so that it can provide trading advice using the predicted prices.s

Preliminary data exploration

The purpose of a data exploration in this context is typically to get a better understanding of the sort of data being used. Some general things like checking for nulls and anomalies as well as more case specific information about its autocorrelation and partial correlation plots and whether it is stationary. This information helps to ensure that we are not training our ML model on junk data and helps to inform the parameters of the ARIMA model.
